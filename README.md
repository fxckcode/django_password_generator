<h1 align="center">Password Generator</h1>
<h3 align="center">By Django</h3>

### Install and Run
```bash
$ git clone https://github.com/fxckcode/django_password_generator.git

$ cd django_passoword_generator

$ pip install django

$ python manage.py runserver
```
